# Resume-Frontend (Premium AI Dashboard)

A high-end, premium frontend for the Resume Analyzer application, built with React, Tailwind CSS, and Framer Motion. Powered by Google Gemini AI (via the Resume-Backend).

## Features
- **Premium Design**: Modern dark theme with glassmorphism and dynamic background effects.
- **Fluid Animations**: High-fidelity transitions and hover effects using Framer Motion.
- **Interactive Uploader**: Sophisticated drag-and-drop interface with real-time feedback.
- **Visual Analytics**: Dynamic ATS score presentation and detailed feedback categorization.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/pranavMattipi/Resume-Frontend.git
   cd Resume-Frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Check Backend Connectivity**:
   Ensure the backend is running at `https://resume-backend-one-mu.vercel.app/`. You can change the API URL in `App.jsx` if necessary.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```

## Tech Stack
- React (Vite)
- Tailwind CSS
- Framer Motion
- Lucide React (Icons)
